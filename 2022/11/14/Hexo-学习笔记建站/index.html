Hexo建站-学习笔记

发表于2022-11-14|更新于2022-11-15|hexo
|阅读量:

需先掌握的知识点：node npm git github gitee Markdown

一：建项目

新建项目文件夹

  1. 安装Hexo:

    npm install hexo-cli

    项目内安装或全局安装都可

    全局安装 npm install hexo-cli -g

  2. 初始化

    hexo init

  3. 导包

    npm install

  4. 打开运行网站

    1. hexo c 清理缓存
    2. hexo g 生成静态文件
    3. hexo d 部署网站(若生成静态文件时使用命令hexo g -d 则不再需要hexo d 具体命令使用参考Hexo命令文档)
    4. hexo s 启动本地服务器

  5. 主题安装更换

    每个主题有不同要求，可在Hexo文档主题内挑选自己喜欢的主题后根据文档安装更换与配置

    Hexo主题文档

  6. 写博客

    在项目文件内source/_posts目录下新建md文件并在md文件内编写博客
