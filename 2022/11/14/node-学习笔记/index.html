初识node

Node.js® 是一个基于 Chrome V8 引擎 的 JavaScript 运行环境

node终端命令

常用命令 作用
node -v 查看所安装的node的版本号
node 文件名称 将文件放入node环境运行

npm包管理工具:

npm包管理工具随着node.js的安装包一起被安装到了用户的电脑上；

[npm官方][https://www.npmjs.com/] 可以搜索想要的包和查看使用方法；

下载包需要去官方服务器https://registry.npm.org/；

也可以去淘宝镜像源下包https://registry.npm.taobao.org/

常用命令 作用
npm -v 查看所安装的npm包管理工具的版本号
npm i 安装所有依赖的包
npm i 包名 安装指定的包的最新版本
npm i 包名@版本号 安装指定的包的指定的具体版本
npm i 包名 -g 在全局安装指定的包
npm i 包名 -D 安装指定的包并记录到devDependencies节点中（devDependencies 开发依赖包， dependencies 核心依赖包）
npm uninstall 包名 卸载指定的包
npm init 初始化
npm init -y 初始化并创建package.json文件
npm config get registry 查看当前下包的镜像源
npm config set registry=镜像源地址 切换当前下包的镜像源
npm login 登入包
npm publish 发布包
npm unpublish 删除包(上传72小时内才可删除)

切换镜像源小工具nrm:

npm i nrm -g全局安装nrm；

nrm ls查看所有可用镜像源；

nrm use 镜像源名 切换镜像源。

模块

模块通常也叫作包，分为node内置模块，第三方模块，自定义模块 三种。

内置模块与第三方模块的引入方式是一样的，都是require('模块名称')；

自定义模块引入方式是require('模块相对路径')相对路径以./或../开头;

引入模块的机制是：若请求的是模块名称，则先从内置找，找不到则找安装的第三方模块(从当前文件目录开始找/node_modules文件夹加载第三方模块，找不到则移动到上一层父目录找直到文件系统的根目录)；若请求的是自定义模块，则按请求路径找。

fs模块与http模块为node内置模块。
